# Installation

## Prerequisites
Obtain:
* Python3
* pip
* [pipenv](https://pipenv.pypa.io/en/latest/) `pip install pipenv`



## Project installation
Install with `--dev` to be able to run tests.
```
pipenv install --dev
```

# Run
```
pipenv run python ./src/main.py 
```

# Tests
```
pipenv run python -m pytest 
```
